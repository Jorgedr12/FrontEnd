# Portafolio Web – Jorge Luis

Este repositorio contiene mi portafolio web personal, realizado como práctica de FrontEnd. En la página se muestra información sobre mí, algunos proyectos y una sección de contacto.

## Descripción

Es una página web estática . Cuenta con secciones de inicio, sobre mí, proyectos y contacto, accesibles desde un menú de navegación.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Contenido del sitio

- Información personal
- Descarga de currículum en formato PDF
- Proyectos realizados
- Formulario de contacto
- Enlaces a GitHub y LinkedIn
